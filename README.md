# simpleWS

Auteur : ant1mcl

## Overview

Ce projet fournit un lanceur simple pour exécuter des scripts Groovy, notamment avec la bibliothèque http-builder-ng pour des requêtes HTTP simples ou avancées. Il permet d'exécuter facilement des scripts Groovy depuis la ligne de commande via un JAR unique.

## Features
- Lance et exécute des scripts Groovy avec prise en charge http-builder-ng
- Exemple de script inclus : `script.groovy`
- Dépendances nécessaires : `groovy`, `groovy-json`, `groovy-xml`, `http-builder-ng-core`, `http-builder-ng-apache`
- Supporte les appels HTTP REST 

## Requirements
- Java 8 or higher
- Maven
- Connexion Internet pour les dépendances

## Build Instructions

1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd playwright-groovy-launcher
   ```
2. Build the project with Maven:
   ```sh
   mvn clean package
   ```
   This will generate a JAR file with dependencies in the `target` directory.

## Usage

Pour exécuter un script Groovy (ex : `script.groovy`) :

```sh
java -jar .\target\simplews-1.0-jar-with-dependencies.jar .\src\main\groovy\org\example\script.groovy
```

Si aucun script n'est fourni, un message d'utilisation s'affichera :

```
Usage: java -jar .\target\simplews-1.0-jar-with-dependencies.jar .\src\main\groovy\org\example\script.groovy
```
